# 🚀 START HERE - Quick Reference

## Start Servers (Copy & Paste)

### Terminal 1 - Backend
```bash
cd backend
uvicorn app.main:app --host 0.0.0.0 --port 8001
```

### Terminal 2 - Frontend
```bash
cd frontend
npm run dev
```
