⏳ UC013: Complete Symptom Questionnaire - SUMMARY BELOW
   - Patient answers guided questions about skin lesion symptoms
   - Questions: duration, pain level, itching, bleeding, size changes, location
   - Responses stored with analysis for doctor review
   - Data used in AI risk assessment

⏳ UC014: AI Lesion Analysis - SUMMARY BELOW  
   - Vision Transformer (ViT) model analyzes validated skin image
   - Detects and localizes lesion in image
   - Classifies into 7 cancer types with confidence scores
   - Processing time: <10 seconds
   - Accuracy: 84% (validated on HAM10000 dataset)

⏳ UC015: Display AI Results - SUMMARY BELOW
   - Shows cancer type classification result
   - Displays confidence percentage (0-100%)
   - Shows risk level (Low/Medium/High/Urgent) with color coding
   - Provides recommended next actions
   - Includes disclaimers about screening vs diagnosis

⏳ UC016: Generate Risk Assessment - SUMMARY BELOW
   - AI assigns risk level based on cancer type + confidence
   - Melanoma or High Confidence (>80%) = Urgent
   - Basal Cell / Actinic Keratoses = High
   - Moderate confidence (50-80%) = Medium
   - Low confidence (<50%) or benign types = Low


⏳ UC017: Save Analysis Report - SUMMARY BELOW
   - System stores complete analysis in database
   - Includes: image, AI predictions, symptom responses, risk level
   - Linked to patient account and health profile
   - Generates unique report ID
   - Creates timestamp for tracking

⏳ UC018: View Analysis History - SUMMARY BELOW
   - Patient accesses all previous skin analyses
   - Sorted chronologically (newest first)
   - Shows date, cancer type detected, risk level
   - Can click to view full report details
   - Can download/print reports

SECTION 4: HOSPITAL FINDER & APPOINTMENTS (5 Use Cases)
================================================================================

⏳ UC019: Search Nearby Hospitals - SUMMARY BELOW
   - Uses patient's GPS location or manual address input
   - Searches via Google Maps API (Google Places)
   - Finds dermatology hospitals + general hospitals
   - Search radius: 50km (31 miles)
   - Returns up to 180 results

⏳ UC020: Filter Hospital Results - SUMMARY BELOW
   - Sorts hospitals by rating (4.5+ stars first), then distance
   - Displays rating stars, distance in km, operating hours
   - Shows "Open Now" or "Closed" status
   - Can filter by: currently open, rating threshold
   - Shows hospital specialties if available


⏳ UC021: View Hospital Details - SUMMARY BELOW
   - Shows comprehensive facility information
   - Address, phone number, website
   - Operating hours (daily schedule)
   - Specialties and services offered
   - Patient reviews and ratings from Google
   - Photos of facility

⏳ UC022: Get Navigation Directions - SUMMARY BELOW
   - Provides turn-by-turn directions to hospital
   - Uses Google Maps integration
   - Shows estimated travel time and distance
   - Multiple route options (fastest, shortest)
   - Can open in Google Maps app on mobile

⏳ UC023: Book Appointment - SUMMARY BELOW
   - Patient selects hospital from search results
   - Patient chooses preferred date and time
   - System attaches analysis report automatically
   - Patient can add notes or concerns
   - System submits appointment request to hospital
   - Hospital staff must confirm (UC027) before appointment is final

SECTION 5: REPORT REVIEW (HOSPITAL STAFF) (4 Use Cases)
================================================================================

⏳ UC024: View Reports by Urgency - SUMMARY BELOW
   - Hospital staff see incoming patient reports on dashboard
   - Organized by risk level: Urgent → High → Medium → Low
   - Color-coded for quick visual identification
   - Shows: Report ID, Date, Cancer Type, Confidence, Risk Level
   - Patient identity hidden until appointment confirmed


⏳ UC025: Review Patient Report - SUMMARY BELOW
   - Staff clicks on report to view details
   - Can view: AI analysis results, skin lesion image, symptom responses
   - Risk assessment and recommended urgency displayed
   - Patient identity remains HIDDEN until appointment confirmed
   - Staff can decide to confirm (UC027) or reject (UC028) appointment

⏳ UC026: Access Patient History - SUMMARY BELOW
   - Available ONLY AFTER appointment is confirmed
   - Staff can view complete medical history
   - Access to health profile: conditions, medications, allergies
   - View previous skin analyses and results
   - Privacy-protected: access is logged for audit trail
