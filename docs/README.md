## Quick Start Guides

### For Patients
1. Read [Patient User Guide](USER_GUIDE_PATIENTS.md) - Getting Started section
2. Create your account at https://skinguard.com
3. Complete your health profile
4. Upload your first image for analysis
5. Find a doctor near you

### For Doctors
1. Read [Doctor User Guide](USER_GUIDE_DOCTORS.md) - Registration section
2. Register at https://skinguard.com/doctor-signup
3. Submit verification documents
4. Wait for admin approval (1-3 business days)
5. Start reviewing patient reports

### For Administrators
1. Read [Admin User Guide](USER_GUIDE_ADMINS.md) - Getting Started section
2. Log in to admin panel at https://skinguard.com/admin
3. Review pending doctor verifications
4. Monitor system health
5. Moderate flagged content

### For Developers
1. Read [API Documentation](API_DOCUMENTATION.md)
2. Read [Deployment Guide](../DEPLOYMENT_GUIDE.md)
3. Set up development environment
4. Test API endpoints
5. Deploy to production
